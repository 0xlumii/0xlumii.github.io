About me.

The story behind the bugs.

I'm 0xlumi — an application security engineer and bug bounty hunter based in Nigeria. I spend my days breaking into applications so that real attackers can't.

My journey into security started with curiosity — taking things apart to understand how they work, then figuring out how they break. That curiosity turned into a career built around finding vulnerabilities that automated tools miss and helping organizations fix them before they become headlines.

I specialize in web application security, API exploitation, AI/LLM security, and mobile app hacking. Whether it's chaining low-severity bugs into critical exploits or auditing the latest AI-powered features, I approach every target with the same mindset: think like an attacker, report like a professional.
